# Version Inventory / Local Data Workbench — Meta Documentation

Этот архив содержит **мета-документацию** проекта:

- системный промпт и манифесты для Cursor/AI-агентов;
- спецификации архитектуры;
- правила разработки, рефакторинга и тестирования;
- пайплайн релизов.

Это **не сам фронтенд-проект**, а его "мозг" и контракт на развитие.

## Структура

- `MANIFEST.md` — общий манифест проекта.
- `docs/` — человеческие спецификации и описания модулей.
- `cursor/` — файлы, которые удобно использовать как системные промпты / конфиги для Cursor.

Эти файлы можно положить в отдельный репозиторий или в каталог `meta/` основного проекта.
